# MyTems

MyTems es un marketplace de artículos virtuales de videojuegos. Permite a los jugadores comprar, vender e intercambiar skins, items y más de juegos como CS:GO, Dota 2 y Rust — de forma rápida, segura y sin complicaciones.

## Páginas

- **Inicio** — hero con artículos destacados, estadísticas, sección de tienda y sobre nosotros
- **Artículos** — catálogo de items y juegos con filtros por categoría
- **Registro** — formulario de creación de cuenta

## Tecnologías

- HTML5 semántico
- CSS3 (custom properties, Flexbox, CSS Grid)
- JavaScript vanilla (menú responsive)
- Google Fonts — Montserrat + Poppins
